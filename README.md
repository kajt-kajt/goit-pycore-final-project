# Amadeus_bot — це консольний бот для керування контактами. Підтримує збереження даних, роботу з телефонами, email, адресами, датами народження та пошук найближчих днів народження.

# Amadeus_bot створений командою №6 для скаладання екзамінаційного проекту в рамках курсу Python.

## Можливості
- Контакти: додати, змінити, видалити; кілька телефонів/e‑mail на один контакт; адреса; повний перегляд контакту.
- Дні народження: зберегти дату, показати дату, список найближчих днів народження за N днів.
- Нотатки: створити нотатку, теги до нотатки, пошук нотаток за тегом, видалення тегів і нотаток.
- Пошук: за ім’ям, адресою, телефоном, e‑mail (`search-contacts <рядок>`).
- Довідка: кольоровий вивід команд у терміналі (`help` або `?`).

## Швидкий старт 

### Windows (через .bat)
1. Відкрийте теку проєкту у Провіднику.
2. Двічі клацніть `setup_run.bat` і дочекайтеся завершення. Скрипт:
   - створить віртуальне середовище в проєкті,
   - встановить залежності,
   - додасть ярлик `assistant` у стандартний шлях.
3. Відкрийте **нове** вікно PowerShell або CMD.
4. У будь-якій теці введіть команду `assistant` і натисніть Enter — бот запуститься.

- Щоб видалити середовище/ярлик/файл даних — запустіть `cleanup.bat`.

### Linux/macOS (через .sh)
1. (Необов’язково) Якщо хочете запускати напряму (`./setup_linux.sh`), надайте право на виконання: `chmod +x setup_linux.sh`.
2. Запустіть: `sh ./setup_linux.sh`. Скрипт створить `.venv`, поставить залежності й одразу запустить бота.

- Очищення: `./cleanup_linux.sh` (видалить `.venv` і `books.pkl`).

### Ручне встановлення (якщо скрипти не підходять)
1. Переконайтеся, що встановлено Python 3.10+ і він доступний у PATH.
2. У корені проєкту створіть середовище та активуйте його:
   - Windows (PowerShell): `python -m venv .venv; .venv\Scripts\activate`
   - Linux/macOS: `python3 -m venv .venv && source .venv/bin/activate`
3. Встановіть залежності: `pip install -r requirements.txt`
4. Запустіть: `python main.py`

## Використання
| Дія | Команда |
| --- | --- |
| Переглянути команди | `help` або `?` |
| Додати контакт | `add Vasyl +380501234567` |
| Замінити телефон | `change Vasyl +380501234567 +380501112233` |
| Додати e‑mail | `add-email Vasyl vasyl@example.com` |
| Додати адресу | `add-address Vasyl Kyiv, Khreshchatyk 1` |
| Додати день народження | `add-birthday Vasyl 31.12.1990` |
| Показати контакт | `show-contact Vasyl` |
| Показати дні народження на 14 днів | `birthdays 14` |
| Створити нотатку | `add-note "купить молоко"` |
| Додати тег нотатці | `add-note-tag <id> urgent` |
| Знайти нотатки за тегом | `notes-by-tag urgent` |
| Пошук по всіх контактах | `search-contacts vasyl` |
| Пошук по нотатках | `search-notes note` |
| Вийти й зберегти | `close` або `exit` |

### Формати вводу
- Телефон: українські номери, приводяться до вигляду `+380XXXXXXXXX`; можна вводити з пробілами/дужками/`00`/`+`.
- E‑mail: звичайний формат `user@domain.tld`.
- Дата народження: `DD.MM.YYYY`.

## Структура проєкту
- `main.py` — CLI-цикл, завантаження/збереження даних, маршрутизація команд.
- `src/entities/` — моделі (Record, AddressBook, Phone, Email, Address, Birthday, Note, NoteBook тощо), валідація введених значень.
- `src/handlers/` — обробники команд (add/change/delete/show/search/notes/help).
- Скрипти автозапуску/очищення: `setup_run.bat`, `assistant.bat`, `cleanup.bat`, `setup_linux.sh`, `cleanup_linux.sh`.
- `requirements.txt` — залежності.
